# spring-completion

Bash completion support for Ruby on Rails Spring commands:

* `help`
* `binstub`
* `status`
* `stop`
* `rails`
* `rake`
* `rspec`

## Installation

### macOS

```bash
brew install joenyland/shell-completions/spring-completion 
```

### Ubuntu/Debian

```bash
cp spring.sh /etc/bash_completion.d/
```

### Other Linux

```bash
source spring.sh
```
